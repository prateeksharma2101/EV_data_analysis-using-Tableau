# ⚡ Electric Vehicle (EV) Data Analysis Dashboard - Tableau

## 📌 Project Overview
This project analyzes Electric Vehicle (EV) data using Tableau to uncover insights into adoption trends, regional distribution, and vehicle categories. The dashboard provides an interactive view of how EV usage is growing across different regions and segments.

---

## 📊 Objectives
- Analyze **EV adoption trends over time**
- Identify **top regions with highest EV usage**
- Understand **distribution across vehicle types**
- Provide insights into **growth patterns and market trends**

---

## 🛠️ Tools & Technologies
- **Tableau** – Data visualization & dashboard creation  
- **Excel / CSV** – Dataset  
- **Data Cleaning & Transformation** – Preprocessing for analysis  

---

## 📂 Dataset
The dataset includes:
- Region / Location  
- Vehicle Type  
- Registration / Adoption Data  
- Time-based trends  

---

## 📈 Key Insights
- 📌 EV adoption is steadily increasing across multiple regions  
- 📌 Certain regions show significantly higher adoption rates  
- 📌 Distribution varies across different vehicle categories  
- 📌 Growth trends indicate rising demand for electric mobility  

---

## 📊 Dashboard Features
- Interactive filters (Region, Vehicle Type, Time)  
- KPI indicators for quick insights  
- Visualizations:
  - EV adoption trends over time  
  - Region-wise distribution  
  - Vehicle category analysis
